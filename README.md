# Sven’s Website

A personal portfolio website created to present my **projects, skills, interests, and learning progress** as an aspiring **Software Developer / Fachinformatiker für Anwendungsentwicklung**.

The website serves as a **learning project** and portfolio where I experiment with **HTML, CSS, JavaScript, UI design, and structured web development**.

---

# Live Website

You can view the website here:

**GitHub Pages:**
https://xshadowhuntx.github.io/svens-website/

---

# Features

### Personal Portfolio

* Introduction and personal background
* Career path and motivation
* Skills and current learning topics

### Projects Section

* Overview of coding projects
* Technology stack descriptions
* GitHub links to project repositories

### Interactive Game Library

* Retro arcade styled page
* Categorized by platform (SNES, GB, PS1, PS2, PS3, PC)
* Dynamic filtering system
* Automatically generated **Top Picks**

### Movie & Series Library

* Large curated list of movies and series
* Genre-based filtering
* Dynamic list generation with JavaScript
* Rating system using star visualization

### Contact Page

* Email contact
* GitHub profile link
* Easy access for recruiters or collaborators

### Dynamic Elements

* Automatic year update in footer
* Automatic age calculation
* Dynamic filtering systems
* Generated ranking lists (Top picks)
